# cc-assignment1

## Dependencies
* [java SE dev kit](https://www.oracle.com/java/technologies/downloads/)
* [maven](https://maven.apache.org/download.cgi)
* [spark](https://spark.apache.org/downloads.html)

## Compiling and running
```console
# compile project
mvn clean package

# ensure script is executable
chmod +x run.sh

# run program
./run.sh
```
